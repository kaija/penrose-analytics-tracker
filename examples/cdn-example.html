Usage

Include the script from CDN:

<script src="https://cdn.example.com/analytics-tracker.min.js"></script>

Initialize and use:

const tracker = new AnalyticsTracker({
  project: 'my-project',
  endpoint: 'https://analytics.example.com'
});

tracker.track('page_view');
